# doxx

A WYSIWYG document editor in the browser.

## Actions:

- [x] Type letters
- [x] Press backspace
- [x] Press enter
- [ ] Press < ^ > v
- [ ] Reach end of line
- [ ] Click within chunk
- [ ] Click within line
- [ ] Select characters
- [ ] Choose B | U | I | font | size
- [ ] Align L | C | R | J
